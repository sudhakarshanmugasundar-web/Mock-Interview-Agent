# 🤖 AI-Powered Mock Interview Agent with Intelligent Resume Analysis and Real-Time Emotion Recognition

An AI-powered web application that helps job seekers prepare for technical and HR interviews through personalized mock interviews, resume analysis, and real-time emotion recognition. The platform provides an interactive interview experience, evaluates candidate performance, and offers actionable feedback to improve interview skills.

---

## 📌 Project Overview

The AI-Powered Mock Interview Agent simulates real interview scenarios by analyzing a candidate's resume, generating role-specific questions, monitoring facial emotions during the interview, and evaluating overall performance.

The system helps students and job seekers improve their communication skills, technical knowledge, confidence, and interview readiness through AI-driven feedback.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Sudhakar-2428/Mock-Interview-Agent.git
```

```bash
cd Mock-Interview-Agent
```

---

## Backend Setup

```bash
cd backend
```

Configure the environment variables.

Run the application:

```bash
./mvnw spring-boot:run
```

---

## Frontend Setup

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Environment Variables

Create the required environment variables before running the project.

### Backend

```
SPRING_DATASOURCE_URL
SPRING_DATASOURCE_USERNAME
SPRING_DATASOURCE_PASSWORD
SPRING_PROFILES_ACTIVE
JWT_SECRET
OPENAI_API_KEY
```
